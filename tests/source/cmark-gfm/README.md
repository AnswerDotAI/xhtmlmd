# cmark-gfm fixtures

`spec.txt` and `extensions.txt` are copied from `github/cmark-gfm` test
fixtures. They are used as upstream conformance inputs for the ignored
`tests/conformance.rs` report.

The fixture files declare the Creative Commons CC-BY-SA 4.0 license in their
front matter. The cmark-gfm repository also carries that notice in `COPYING`.
